# MAX Loyalty Bot

Версия проекта программы лояльности.

## Что реализовано
- бонус за оформленный полис как процент от суммы полиса;
- бонус активируется с даты начала полиса;
- срок действия бонуса заканчивается в дату окончания полиса;
- бонусы суммируются по нескольким полисам;
- списание бонусов только на добровольные виды страхования;
- заявка на вывод на карту при пороге от 1000 рублей;
- реферальный бонус за приглашённого клиента;
- уровень лояльности по общей сумме полисов за период 365 дней;
- webhook API для подключения бота и административных действий.

## Быстрый запуск локально
```bash
python -m venv .venv
source .venv/bin/activate  # Linux/macOS
# .venv\Scripts\activate   # Windows
pip install -r requirements.txt
cp .env.example .env
uvicorn bot_app:app --reload
